## 5. 制作指南针
### 1). 新建一个图层，并绘制一个多边形
::: tip 温馨小提示
按住鼠标左键同时，按下键盘方向键<kbd>上|下</kbd>调整多边形边数
:::
### 2). 调整三角形底边和高的弧度,拖拉三角形三个顶角弧度至合适大小，并适当旋转三角形
::: tip 贴心小提示
可以适当拉取多条参考线，来平衡图标位置
:::
### ![Pic1](https://cdn.jsdelivr.net/gh/OtoyaLee/imgpicgo/20220421135529.png)
